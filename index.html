Projetos

Movie Mania

Movie Mania é um aplicativo de com informações de filmes desenvolvido com React Native na plataforma expo.dev.

O aplicativo foi desenvolvido com o objetivo de aprender sobre navegação entre telas.

Durante o projeto foi colocado em prática a navegação entre diferente telas utilizando View e useState, a renderização de textos e imagens com Text e Image, interação com botões através de eventos com TouchableHighlight e a estilização das páginas com StyleSheet.

Veja o projeto completo.

Conversor de Temperatura

Este é um aplicativo de conversão de temperatura desenvolvido com React Native na plataforma expo.dev.

O aplicativo foi desenvolvido com o objetivo de aprender sobre inserção de dados com o componente Input.

Durante o projeto foram utilizados o Input para permitir que o usuário digite uma temperatura, funções para coverter uma temperatura em outra a partir dos dados inseridos, eventos para tornar o Button interativo e useState para atualizar Text com o resultado da conversão.

Veja o projeto completo.

Megaware

Este é um aplicativo de e-commerce desenvolvido com React Native na plataforma expo.dev.

O aplicativo foi desenvolvido com o objetivo de aprender sobre a redenrização de items com o componente FlatList.

Durante o projeto foram utilizados o Input e RadioButton (componente personalizado que desenvolvi seguindo um tutorial) para permitir que o usuário filtre o conteúdo que será exbido na página. Além disso, o usuário pode navegar para a página de um produto específico, adicionar e remover produtos do carrinho com o componente Button.

Veja o projeto completo.

Comidinhas

Este é um aplicativo de receitas desenvolvido com React Native na plataforma expo.dev.

O aplicativo foi desenvolvido com o objetivo de aprender sobre navegação entre páginas com o componente NavigationContainer.

Neste projeto os componentes Stack.Navigator e Stack.Screen foram utilizados para criar a nevagecação. Além disso, foi compreendido como as páginas podem ser colocadas em arquivos separados para melhorar a organização do projeto. Também foi visto como personalizar o cabeçalho e passar dados entre páginas utilizando JSON. Por fim, o componente ActivityIndicator foi utilizado para criar uma tela de carregamento.

Veja o projeto completo.
